uni-helper was originally just a personal project, but has now become an unofficial non-profit organization.

uni-helper provides a number of packages that enhance the development experience of the uni-app family of products, some of which I'm trying to maintain, including [TypeScript type definitions](https://github.com/orgs/uni-helper/repositories?q=types& type=all&language=&sort=), VSCode plugins, and more.

tailwind-extensions

tailwind-extensions is a library that extends tailwindcss.

It provides more configuration values to reduce the use of inferred values, which brings tailwindcss one step closer to unocss.

In addition, it provides presets for use with antd, antdv, element-plus, making tailwindcss easier to integrate with these UI libraries.

It also provides miniprogream and easy presets to make tailwindcss more friendly to the Chinese ecosystem.
