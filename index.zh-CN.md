## 介绍

我在 2018 年开始接触并自学前端，2019 年 6 月从中山大学软件工程毕业，获得统招本科学历学士学位。

我是一名前端工程师，具有扎实的编程基础、严格的代码要求和丰富的项目经验。我熟练掌握原生 JavaScript / TypeScript、Vue 和 React 等相关前端技术，能够独立完成前端项目的开发，并与团队成员协作完成复杂项目的开发。

在我的职业生涯中，我参与了多个项目的开发，涉及的领域包括餐饮、教育、商务、Web 3 等。我在这些项目中承担起不同的角色，包括前端开发、全栈开发、前端组长等。在这些项目中，我积累了丰富的经验，熟悉项目开发的流程和要求。我对于代码的质量要求很高，善于使用各种工具和技术手段提高代码的可维护性和可读性。

除了工作之外，我还研究和应用开源技术，我在 Github 上有多个开源项目，并定期在个人站上发布技术博客和文章。我相信开源技术的力量，也相信通过分享和交流可以不断提高自己的技术水平。

作为一名前端工程师，我乐于与产品经理协作，致力于提高项目的用户体验和性能。我也关注项目的稳定性和可用性，能够使用各种工具和技术手段保证项目的质量和稳定性。在未来，我希望成为一名基于 TypeScript 的、兼具产品、运维能力和思维的 10x 全栈工程师。

## 项目

此处仅列出部分项目，更多可查看 GitHub。

### [fabric](https://github.com/ModyQyW/fabric)

这是一个开箱即用的共享规范，适用于不同的 JavaScript / TypeScript 项目，包括了 Git、EditorConfig、tsconfig、Prettier、ESLint、Stylelint、Markdownlint、Commitlint、Conmitizen、Publint、LintStaged、Husky、SimpleGitHooks、VScode 等工具的建议配置和使用。

### [vite-plugin-eslint2](https://github.com/ModyQyW/vite-plugin-eslint2)

这是一个用于 vite 的 ESLint 插件，与 eslint-webpack-plugin 作用类似。最初因为 vite-plugin-eslint 维护不够活跃而分叉出来维护并添加新功能。

尽管我仍然在维护它，但我不建议再去使用它。

### [vite-plugin-stylelint](https://github.com/ModyQyW/vite-plugin-stylelint)

这是一个用于 vite 的 Stylelint 插件，与 stylelint-webpack-plugin 作用类似。截至 2023-06-09，周下载量有 13k。基于这个插件的经验，我为 [vite-plugin-checker](https://github.com/fi3ework/vite-plugin-checker) 贡献了 [PR](https://github.com/fi3ework/vite-plugin-checker/pull/158)。

尽管我仍然在维护它，但我不建议再去使用它。

### uni-helper

uni-helper 本来只是一个 [个人项目](https://github.com/ModyQyW/uni-helper)，但现在已经成为一个 [用爱发电的非官方非盈利组织](https://github.com/uni-helper)。

uni-helper 提供了大量增强 uni-app 系列产品的开发体验的包，我在尽力维护其中一部分，包括 [TypeScript 类型定义](https://github.com/orgs/uni-helper/repositories?q=types&type=all&language=&sort=)、[VSCode 插件](https://github.com/orgs/uni-helper/repositories?q=vscode&type=all&language=&sort=) 等。

截至 2023-06-09，VSCode 插件累计下载量有 177k，我相信大部分使用 VSCode 的 uni-app 开发者都使用过或正在使用我开发的 VSCode 插件和 TypeScript 类型定义。

### [tailwind-extensions](https://github.com/ModyQyW/tailwind-extensions)

tailwind-extensions 是一个拓展 TailwindCSS 的库。

它提供了更多的配置值以减少推断值的使用，这一点使 TailwindCSS 向 UnoCSS 更靠近一步。它还提供了和 AntDesign、ElementPlus 的结合使用预设，使 TailwindCSS 更容易和这些 UI 库集成使用。考虑到国内生态，它还提供了小程序和适老化预设，使 TailwindCSS 对国内生态更友好。

我另外也参与了 [unocss-preset-antd](https://github.com/Dunqing/unocss-preset-antd) 和 [unocss-preset-element-plus](https://github.com/ModyQyW/unocss-preset-element-plus) 的开发。
